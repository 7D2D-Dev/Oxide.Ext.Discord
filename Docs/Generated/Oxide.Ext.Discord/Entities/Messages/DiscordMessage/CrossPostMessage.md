# DiscordMessage.CrossPostMessage method (1 of 2)

Crosspost a message in a News Channel to following channels. This endpoint requires the 'SEND_MESSAGES' permission, if the current user sent the message, or additionally the 'MANAGE_MESSAGES' permission, for all other messages, to be present for the current user. See [Crosspost Message](https://discord.com/developers/docs/resources/channel#crosspost-message)
